Network Infrastructure Support

Enterprise network infrastructure configuration and management for improved security and performance

Project Overview

Designed and implemented a comprehensive network infrastructure solution for enterprise environment, focusing on L2/L3 switch operations, VLAN configuration, and Cisco Meraki firewall deployment to enhance network security and performance.

Problems Solved

  • Network segmentation issues causing security vulnerabilities
  • Poor network performance due to broadcast storms
  • Lack of centralized firewall management
  • Inefficient traffic routing between departments
  • Limited network monitoring and visibility

Implementation Steps

Step 1: Network Assessment

Conducted comprehensive network audit to identify bottlenecks and security gaps

Step 2: VLAN Design

Designed VLAN structure for departments: Admin (VLAN 10), IT (VLAN 20), Guest (VLAN 30)

Step 3: Switch Configuration

Configured L2/L3 switches with inter-VLAN routing and trunk ports

Step 4: Firewall Deployment

Deployed Cisco Meraki MX firewall with security policies and content filtering

Step 5: Testing & Optimization

Performed network testing and optimization for optimal performance
